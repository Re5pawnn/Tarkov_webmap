# TarkovMapLocator 使用说明

TarkovMapLocator 是一个用于《逃离塔科夫》的地图定位工具。电脑端负责读取截图和日志，手机端可以通过浏览器查看地图位置和队友位置。

## 下载与安装

1. 打开项目的 GitHub Releases 页面。
2. 下载最新版本里的 `TarkovMapLocator.exe`。
3. 双击安装包。
4. 按安装向导选择安装路径。
5. 如需桌面入口，勾选“创建桌面快捷方式”。
6. 安装完成后，从桌面快捷方式或开始菜单启动。

## 首次使用

启动后会自动打开浏览器页面。

在电脑端页面中依次完成：

1. 点击“选择截图目录”，选择游戏截图保存目录。
2. 点击“选择Log目录”，选择游戏安装目录或 `Logs` 目录。
3. 点击“开始监听”。
4. 进入游戏后按截图键，地图上会显示你的位置和朝向。

目录选择会自动记住。下次启动后，如果浏览器允许读取权限，可以直接继续使用；如果浏览器提示授权，按提示允许即可。

## 地图功能

页面上方可以选择：

- 地图：手动切换地图，或保持自动选择。
- 底图：在支持的地图上切换抽象图和卫星图。
- 点位显示：展开后选择要显示的点位类型。

手机端默认只显示适合查看的内容，界面会隐藏电脑端监听和任务相关控件。

## 手机访问

手机和电脑需要能访问同一个地址。

常见用法：

1. 在电脑上启动 TarkovMapLocator。
2. 用 Sakura Frp 或其他内网穿透工具，把电脑上的网页端口转发出去。
3. 在手机浏览器打开穿透后的网页地址。
4. 手机会自动进入简化页面，只显示地图、点位和队友同步信息。

如果手机页面提示地图元数据加载失败，通常是网络不稳定或穿透地址不可用。刷新页面或检查穿透服务是否正常。

## 队友同步

队友同步适合多人组队时查看彼此位置。

房主操作：

1. 在电脑端打开“队友同步”区域。
2. 设置显示名称和颜色。
3. 保持“房主模式”。
4. 点击“开启同步”。
5. 使用 Sakura Frp 建立一条 TCP 隧道，转发本机同步端口。
6. 把 Sakura Frp 生成的远端地址发给队友。

队友操作：

1. 打开工具。
2. 在“队友同步”区域点击“切换到连接模式”。
3. 粘贴房主发来的 Sakura Frp 地址，例如 `example.com:12345`。
4. 点击“开启同步”。

同步成功后，地图上会显示队友位置，列表中也会显示队友状态。

## Sakura Frp 建议

通常需要两条隧道：

- 网页访问隧道：用于手机浏览器打开地图页面。
- 队友同步隧道：用于队友之间同步位置。

网页访问隧道转发工具启动后显示的网页端口。

队友同步隧道转发默认同步端口：

```text
39247
```

请把 Sakura Frp 提供的完整远端地址复制到工具里，不要只复制域名。

## 常见问题

### 启动后没反应

可能是端口被占用或运行环境异常。工具会自动尝试换端口。如果仍失败，会弹出错误提示，并在本机用户目录下写入启动日志。

### 手机访问提示不安全或打不开

如果 Sakura Frp 要求 HTTPS，请使用它提供的 HTTPS 地址。不要把 HTTP 地址强行改成 HTTPS。

### 地图上没有位置

请确认：

- 已选择正确的截图目录。
- 已点击“开始监听”。
- 游戏截图文件名中包含坐标。
- 当前地图选择正确，或保持自动选择。

### Log目录无法识别

可以直接选择游戏安装目录，也可以选择游戏目录下的 `Logs` 文件夹。

### 切换卫星图后没有图片

当前只有已打包的地图支持本地卫星图。没有卫星图资源的地图会继续显示抽象图。

## 卸载

在 Windows“设置 > 应用”中找到 TarkovMapLocator，选择卸载即可。
