# 塔科夫地图定位  
## 所需环境  
- Windows 10/11
- Python 3.10+（内置了python 但最好还是安装一个）
- 浏览器需支持File System Access API
- 樱花穿透（如果你想共享位置的话)
## 地图使用方法  
- exe安装
- 安装后会在桌面创建快捷方式
- 双击自动拉起浏览器
- 选择截图目录（一般为C:\Users\Administrator\Documents\Escape from Tarkov\Screenshots）
- 选择游戏log目录（一般为Escape from Tarkov\build\Logs）
- 点击开始监听
- 按游戏内截图键截图 等待一秒位置即显示在地图上
## 日志功能  
- 会监听并分析日志 显示战局信息（没什么用但看着很舒服嘿嘿）
## 共享位置  
- 双方都运行工具
- 房主运行樱花穿透 建立一条TCP通道 本地端口选择命令提示符里显示的端口 然后在网页打开同步
- 点击日志 复制地址和端口给队友
- 队友在网页填写远端地址和端口（即房主给的地址和端口 注意区分ip和端口号）
- 双方即可共享位置 截图一次更新一次
